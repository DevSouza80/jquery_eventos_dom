# 📚 Estudos de jQuery - Eventos e Interações

Este repositório contém meus estudos iniciais com **jQuery**, focando em eventos do navegador e manipulação de elementos.

## 🚀 Conteúdo estudado

### 📌 Eventos do Window
- `scroll()` → Executa quando o usuário rola a página
- `resize()` → Executa quando a tela é redimensionada

### 📌 Controle de execução
- `setTimeout()` → Executa uma função após um tempo
- `clearTimeout()` → Cancela o timeout anterior (evita múltiplas execuções)

### 📌 Redirecionamento
- `location.href` → Redireciona o usuário para outra página

### 📌 Eventos de clique
- `.click()` → Detecta clique em elementos

### 📌 Controle de comportamento
- `preventDefault()` → Impede comportamento padrão (ex: link)
- `return false` → Alternativa para bloquear ação padrão
- `stopPropagation()` → Impede propagação do evento

### 📌 Manipulação de CSS
- `.css()` → Altera estilo de elementos via jQuery

---

## 🧠 O que aprendi

- Como reagir a ações do usuário (scroll, resize, click)
- Como controlar eventos para evitar comportamentos indesejados
- Como manipular elementos dinamicamente
- Como trabalhar com temporizadores em JavaScript

---

## 🧪 Exemplo prático

- Ao redimensionar a tela → redireciona após 1 segundo
- Clique em `.box` → impede propagação
- Clique no `body` → altera cor da `.box`
